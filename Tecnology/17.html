Navegação na WWW Navegar na Web é muito simples e, normalmente, começa-se por visitor um URL conhecido. Vamos ver um exemplo prático. Um utilizador deseja saber as últimas notícias do Governo de Timor-Leste e sabe que o URL para o respetivo sítio Web é http://timor-leste.gov.tl. Para dar início à navegação é necessário: • abrir o navegador Web. No Edubuntu, por defeito, tens o Firefox disponível. Pode ser chamado a partir do ícone do Firefox ( ) que está disponível na barra de topo do sistema operativo ou através do menu Aplicações -> Internet -> Firefox Navegador Web; • introduzir na caixa de endereço o URL pretendido (http://timor-leste. gov.tl) e carregar na tecla Enter; • dependendo da velocidade de acesso à Internet, a página inicial do Governo de Timor- Leste poderá demorar poucos segundos até alguns minutos a aparecer no navegador Web. Aguarda que a página esteja carregada para começar a ver as últimas notícias do Governo de Timor-Leste. Os navegadores Web têm barras de scroll (deslocamento) que, tal como noutras aplicações que já conheces, permitem deslocar os conteúdos da página na vertical e na horizontal. Quando se passa o cursor pelo título de uma notícia (ou pelo texto “ver mais”) o formato do cursor é alterado. Este comportamento indica que estamos perante uma ligação de hipertexto!
