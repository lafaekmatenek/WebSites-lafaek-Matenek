• a nova página começa a ser carregada pelo navegador Web Para regressar à página anterior só é necessário clicar no botão “Retroceder” ( ) da barra de navegação.Os conteúdos de uma página podem conter vários elementos multimedia e também vários elementos de interação. Por exemplo, neste sítio do Governo de Timor-Leste podemos encontrar: • Menus de Seleção semelhantes aos de uma aplicação, cujas opções só são visíveis quando o cursor se encontra sobre eles; • Formulários que permitem introduzir dados para realizar uma qualquer operação na página. Neste exemplo, temos um formulário de pesquisa no sítio Web; • ligações de configuração que permitem alterar o aspeto ou configurações da página. Neste caso, permitem alterar a língua utilizada no sítio Web. Estes são apenas alguns exemplos de elementos de interação que podem existir numa página Web. Com a prática vais encontrar outros elementos e perceber como funcionam. Favoritos e historic Quando navegamos na Web é normal encontrar algumas páginas que gost aríamos de guardar paramais tarde visitar. No Firefox, os URL favoritos são designados por Marcadores. Noutros navegadores poderão chamar-se Favoritos ou, em inglês, Bookmarks. No Firefox, um marcador para uma página Web pode ser adicionado através do menu Marcadores -> Adicionar marcador... (esta opção também pode ser ativada através das teclas Crtl + D). Os marcadores guardados ficam armazenados e podem ser consultados de diversos modos. Por exemplo: • no menu Marcadores -> Marcadores recentes; • na Biblioteca do Firefox acessível a partir do menu Marcadores -> Mostrar todos os marcadores;
