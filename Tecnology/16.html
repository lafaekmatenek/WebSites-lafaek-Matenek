Pode ser chamado a partir do ícone do Firefox ( ) que está disponível na barra de topo do sistema operativo ou através do menu Aplicações -> Internet -> Firefox Navegador Web; • introduzir na caixa de endereço o URL pretendido (http://timor-leste. gov.tl) e carregar na tecla Enter; • dependendo da velocidade de acesso à Internet, a página inicial do Governo de Timor- Leste poderá demorar poucos segundos até alguns minutos a aparecer no navegador Web. Aguarda que a página esteja carregada para começar a ver as últimas notícias do Governo de Timor-Leste. Quando se passa o cursor pelo título de uma notícia (ou pelo texto “ver mais”) o formato do cursor é alterado. Este comportamento indica que estamos perante uma ligação de hipertexto! Clicando nessa ligação podemos verificar que: • o URL da página muda automaticamente; • a nova página começa a ser carregada pelo navegador Web. Para regressar à página anterior só é necessário clicar no botão “Retroceder” ( ) da barra de navegação. Os conteúdos de uma página podem conter vários elementos multimedia e também vários elementos de interação. Por exemplo, neste sítio do Governo de Timor-Leste podemos encontrar: • menus de seleção semelhantes aos de uma aplicação, cujas opções só são visíveis quando o cursor se encontra sobre eles; • formulários que permitem introduzir dados para realizar uma qualquer operação na página. Neste exemplo, temos um formulário de pesquisa no sítio Web; • ligações de configuração que permitem alterar o aspeto ou configurações da página. Neste caso, permitem alterar a língua utilizada no sítio Web. Estes são apenas alguns exemplos de elementos de interação que podem existir numa página Web. Com a prática vais encontrar outros elementos e perceber como funcionam.
