1.2.4. Introdução a uma aplicação de navegação na WWW Caraterísticas Os navegadores Web são aplicações informáticas que, embora variando de aspeto visual, mantêm caraterísticas comuns entre si, nomeadamente: 1. Menu da aplicação: tal como nas outras aplicações informáticas um navegador tem um menu da aplicação onde é possível encontrar todasas funcionalidades da aplicação; 2. Área de separadores (“Tabs” em inglês): um navegador Web permite navegar em várias páginas ao mesmo tempo. Os separadores podem ser adicionados livremente e, cada um, permite carregar uma página Web; 3. Barra de navegação: a barra de navegação contém os elementos fundamentais para navegar na web: • Setas de navegação: permitem navegar entre as páginas visitadas num separador. Por exemplo, após seguir uma ligação de hipertexto é possível voltar à página anterior através da seta “Retroceder” ou “Recuar” (“Back” em inglês); • Caixa de endereço (URL): permite especificar (introduzindo o URL) para onde se pretende navegar. Quando navegamos entre páginas, o campo é automaticamente atualizado de modo a mostrar o URL da página que está a ser visualizada no momento; • Caixa de pesquisa: mais à frente iremos ver a importância que a pesquisa tem na Web. Esta caixa de pesquisa está presente em todos os navegadores e serve de atalho para os principais motores de pesquisa existentes; • Botão de página inicial: todos os navegadores têm uma página inicial que é apresentada quando a aplicação é aberta. Este botão leva o navegador para a sua página inicial, que pode ser definida pelo utilizador. 4. Área da página: esta é a área no navegador onde as páginas Web são mostradas e onde o utilizador pode interagir com o seu conteúdo.
